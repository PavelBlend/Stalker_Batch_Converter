# Batch Converter

## Описание
Скрипт написан на Python. Для запуска нужно установить интерпретатор версии 3.0.0 или выше (`www.python.org`). Предназначен для автоматизации конвертирования X-Ray Engine файлов с помощью конвертера от `Bardak'а`.

## Использование
Скопировать `converter.exe` за 22 марта 2013 (более ранние работать не будут) в папку со скриптом. Запустить `batch_converter.py`. После этого создадутся папки `input` и `output`, если их не было. Далее нужно скопировать файлы `*.ogf`, `*.omf` или `*.dm` в папку `input` и нажать необходимые кнопки для конвертирования. Сконвертированные файлы сохранятся в папку `output` и в каталоге со скриптом сохраниться лог файл. Поддерживается ковертация файлов из вложенных каталогов, например, `input\test_dir\test.ogf` сконвертируется в `output\test_dir\test.object`

## Режимы конвертации
ogf - object

ogf - bones

ogf - skls

ogf - skl

omf - skls

omf - skl

dm  - object
