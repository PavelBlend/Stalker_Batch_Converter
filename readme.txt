Описание

STALKER Batch Converter - это скрипт, предназначенный для автоматизации
конвертирования X-Ray Engine файлов с помощью конвертера от Bardak'а. Написан
на Python (http://www.python.org/). Для запуска нужно установить интерпретатор
версии 3.0.0 или выше.

Конвертер взят отсюда:
https://github.com/abramcumner/xray_re-tools
